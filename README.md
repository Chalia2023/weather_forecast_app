# 36小時天氣預報APP
串接中央氣象局api，提供36小時臺灣天氣預報的Flutter APP

## 摘要
- 在 Android 和 iOS 模擬器上皆可運行
- State Manager: riverpod (2.0.0)
- 串接 API: dio (5.4.3+1)

# Tools
- Dart: 3.4.0
- DevTools: 2.34.3
